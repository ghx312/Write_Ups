# Challenge Description
A CTF is a safe space for people to practice security without getting into any legal trouble. Most challenges mimic real world vulnerabilities/practices.

In the web category, challenges typically come in the form of web applications containing sensitive data, with one or more attack vectors present that will allow you to extract the data. This data will be in the form of a flag, in this case with format grey{...}.
